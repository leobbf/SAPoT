Descrição detalhada

Payload para cadastro de novo cliente.

Payload emitido por um novo cliente que tem o intuito de se cadastrar no banco de dados da Central. Organiza-se em: 16 bits definidores do tipo de cliente, 8 bits para quantidade de sensores que operam neste cliente, 8 bits para quantidade de atuatores que operam neste cliente, um vetor com posições de 16 bits para identificar os tipos de sensores e atuadores que operam neste cliente.

◆ clientType

uint16_t SAPoTMessage_registration::clientType

Tipo de Cliente:
0x00: Interface do Usuário (USR)
0x01: Sistema de Monitoramento e Controle para Ambientes Internos (SMCAI)
0x02: Sistema de Monitoramento e Controle para Ambientes Externos (SMCAE)
0x03: Sistema de Monitoramento de Rede Elétrica (SMRE)
0x04: Sistema de Monitoramento de Rede Hidráulica (SMRH)
0x05: Sistema de Reconhecimento Facial (SRF)
0x06 a 0xFF: Reservado para uso futuro.

uint16_t* SAPoTMessage_registration::sensorActuatorType

Ponteiro identificador dos tipos de sensores e atuadores em operação no Cliente:
0x0000: Sensor de Temperatura (TMP)

0x0003: Sensor de detecção de presença (DTP)
0x0004: Câmera de captação de imagens (CAM)
0x0005: Sensor de tensão em rede elétrica CA (VCA)
0x0006: Sensor de corrente em rede elétrica CA (ICA)
0x0007: Sensor de potência em rede elétrica CA (PCA)
0x0008: Sensor de fator de potêncial em rede elétrica CA (FP)
0x0009: Sensor de consumo energético em rede elétrica CA (KWH)
0x000A a 0xFFFF: reservado para uso futuro.
