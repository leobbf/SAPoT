Estruturas de Dados
Aqui estão as estruturas de dados, uniões e suas respectivas descrições:
 CSAPoTCentralPrincipal estrutura para operar uma Central SAPoT
 CSAPoTCentral_create_optionsEstrutura para definir as opções de criação de uma Central SAPoT
 CSAPoTMessage_accessEstrutura para acesso à informação dos clientes cadastrados
 CSAPoTMessage_actuatorDrivePayload para acionar um atuador específico em um Cliente cadastrado
 CSAPoTMessage_headerCabeçalho fixo das mensagem SAPoT
 CSAPoTMessage_modificationPayload para etiquetagem de clientes no banco de dados da Central
 CSAPoTMessage_recordEstrutura para acessar as de informação enviadas pelo cliente, dentre as quais: Dados dos Sensores, Status dos Atuadores e possíveis erros de operação
 CSAPoTMessage_registrationPayload para cadastro de novo cliente
 CSAPoTMessage_solicitationPayload para solicitar o acionamento ou sensoriamento de algum cliente cadastrado no banco de dados
