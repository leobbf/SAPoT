Descrição detalhada

Payload para solicitar o acionamento ou sensoriamento de algum cliente cadastrado no banco de dados.

Payload emitido pelo usuário e recebido pela central. Tem o intuito de solicitar o acionamento de um atuador ou requisitar os dados provenientes de um sensor que estejam operando em algum cliente cadastrado no banco de dados da Central. São 17 bytes divididos em: 11 bytes referentes a etiqueta do cliente cadastrado no banco de dados da Central, 16 bits para identificar o sensor ou o atuador a ser acionado, 16 bits para o temporizador de acionamento do atuador ou para o intervalo de tempo em que o cliente deve enviar as informações do sensor pra a central e os ultimos 16 bits referentes à intensidade de atuação a qual o atuador será submetido.
