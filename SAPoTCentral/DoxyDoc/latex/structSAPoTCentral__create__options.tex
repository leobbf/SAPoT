\doxysubsection{Descrição detalhada}
Estrutura para definir as opções de criação de uma Central S\+A\+PoT. 

Essa estrutura armazena as informações pertinentes aos serviços de transmissão e banco de dados. Na atual versão de protocolo, a Central S\+A\+PoT tem suporte apenas para o protocolo M\+Q\+TT e S\+QL.
