\doxysubsection{Descrição detalhada}
Estrutura para acessar as de informação enviadas pelo cliente, dentre as quais\+: Dados dos Sensores, Status dos Atuadores e possíveis erros de operação. 

A partir de uma solicitação do Usuário à Central que posteriormente é encaminhada ao Cliente, esse emite uma mensagem de resposta para a Central com as informações pertinentes à instrução recebida\+: 0x01\+: Requsição dos dados de todos os sensores (Solicitation) ~\newline
0x02\+: Requisição dos dados de um sensor em específico (Solicitation) ~\newline
0x03\+: Acionamento de um atuador em específico (Solicitation) ~\newline
 ~\newline
 Ou em casos de erro em alguma operação, o Cliente enviará uma mensagem de log para a central com a instrução 0x07. Dessa maneira, a Central utiliza da estrutura \mbox{\hyperlink{structSAPoTMessage__record}{S\+A\+Po\+T\+Message\+\_\+record}} para interpretar as informações contidadas No pacote enviado pelo Cliente.
